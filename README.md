# CleanFresh Laundry

Website landing page modern untuk bisnis laundry yang dirancang untuk meningkatkan kepercayaan pelanggan, menampilkan layanan secara profesional, dan mempermudah proses pemesanan melalui WhatsApp.

## 🌐 Demo Website

https://clean-fresh-laundry.vercel.app/ 

---

# 📖 Tentang Project

CleanFresh Laundry adalah website landing page responsif yang dibuat untuk membantu bisnis laundry mempromosikan layanan mereka secara online.

Website ini menampilkan informasi layanan, harga paket, keunggulan bisnis, testimoni pelanggan, serta kontak yang mudah diakses oleh calon pelanggan.

Desain website mengusung tema kebersihan dan kesegaran dengan dominasi warna biru muda yang identik dengan air dan laundry profesional.

---

# ✨ Fitur Utama

### 🏠 Hero Section

* Headline menarik
* Call To Action (CTA)
* Tombol WhatsApp langsung
* Gambar hero profesional

### ⭐ Keunggulan Bisnis

Menampilkan nilai jual utama:

* Gratis Antar Jemput
* Garansi Tepat Waktu
* Deterjen Premium
* Sistem 1 Mesin 1 Pelanggan
* Operasional Setiap Hari

### 💰 Paket & Harga

Menampilkan berbagai layanan:

* Cuci Kiloan Premium
* Cuci Kiloan Express
* Deep Clean Sepatu / Tas / Helm

Setiap paket memiliki tombol pemesanan langsung menuju WhatsApp.

### 💬 Testimoni Pelanggan

Menampilkan review pelanggan dengan:

* Nama pelanggan
* Rating bintang
* Avatar pelanggan
* Tampilan modern ala Google Review

### 📱 Integrasi WhatsApp

Calon pelanggan dapat langsung menghubungi bisnis melalui tombol WhatsApp yang tersedia pada berbagai bagian website.

### 🗺️ Google Maps

Menampilkan lokasi bisnis melalui Google Maps Embed sehingga pelanggan lebih mudah menemukan lokasi usaha.

### 📞 Informasi Kontak

* Nomor telepon
* Email
* Website
* Alamat lengkap
